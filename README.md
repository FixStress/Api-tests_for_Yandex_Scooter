# Api тесты для сервиса [Яндекс Самокат](https://qa-scooter.praktikum-services.ru/)

## Задачи проекта
1. _**Протестировать эндпоинты для**_
* Создания курьера
* Авторизации курьера
* Удаления курьера
* Создания заказа
* Принятия заказа
* Списка заказов
* Получения заказа по его номеру 
2. _**Построение отчёта при помощи Allure**_
## Инструменты
* Java 11
* Maven
* JUnit 4
* REST Assured
* Allure
* Gson
